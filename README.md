An immersive browser-based AR hand tracking experience built with JavaScript, Canvas API, Web Audio API, and MediaPipe Hands.

Experience neon particle trails, gesture-triggered shockwaves, matrix rain backgrounds, cyberpunk visual effects, and reactive spatial audio — all directly in the browser with no installation required.

  Real-time hand tracking using MediaPipe Hands
  Dual-hand interaction system
  Neon glowing fingertip effects
  Particle physics engine
  Shockwave ripple effects
  Matrix rain animated background
  Gesture detection
